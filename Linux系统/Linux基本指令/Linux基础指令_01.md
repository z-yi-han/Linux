## 指令
### 1. **ls**
显示当前目录(当前用户所处的)下的文件名(文件属性的一种)，空文件夹就是什么都没有,默认只会显示文件名
#### ls -l
ls是命令,-l是命令行选项,中间的空格不能丢,选项可以是0个或多个,多个选项必须用空格隔开,会显示更多**属性**,ls-l一表示目录(列表)(d开头)    
ls -l指定目录,打印指定目录的内容
#### ls -a
a表示all,ls -a表示全部文件(包括隐藏的)
#### -a -l 结合
ls -a -l or ls -l -a or ls -al or ls -la均可
#### -l -a -d
表示只看(指定)目录本身的属性,不会进去，因为目录本身也是文件
### 2.whoami
显示当前用户是谁
### 3.pwd
显示一个用户当前所处的目录
### 4.touch
- 新建一个**普通文件** :touch hello.txt
- 不能创建文件夹
- 更新文件时间，touch一下存在的文件即可
### 5.stat
用来查看文件的三种时间(A  C  M)
### 6.cd
更改目录。例如:cd /root/113,让自己进行路径切换,cd..回退到上级路径  
-  cd ~ # ~ :当前用户自己的家目录。
- cd - : -表示最近所处的路径(来回跳转)
## 知识点
- 目录＝文件夹
- root账号:管理员
- 文件=内容+属性
- Linux区分文件用显示的第一个字母表示
- 第一个字符以d开头表示目录
- 第一个字符以-开头的是普通文件
- 在Linux中,以**“.”** 开头的文件是隐藏文件
- win:“\”是路径分隔符  Linux:"/"是路径分隔符 两个路径分隔符之间一定是目录,最终可能是目录也可能是普通文件
- 在Linux当中,任何目录(包含空)默认系统都会带"."".."目录，其中 **"."** 表示当前目录,**".."** 表示上级目录
- 整个路径最开始的"/"叫做根目录
- Linux整体的文件结构是一颗从根目录/ 开始的一个多叉树，**这棵树的叶子节点一定是普通文件or空目录,非叶子节点一定是非空目录**
- 路径:Linux访问一个文件，先找到他。找到一个文件:路径
- 绝对路径:/home/test,从头开始全写出来
- 相对路径:以非根目录为起始参照位置(比如通过..访问上级然后可以找到同级)
- Linux中任何一个目录都具备唯一性
- 任何一个Linux用户用特定用户名登录系统时默认在家目录下(普通用户)
- 命令行比较接近OS
